A Firefox extension that removes YouTube Shorts and redirects Shorts links to normal videos.

## Features

- Hides Shorts in feed, sidebar, and search results
- Redirects `/shorts/` URLs to standard videos
- Works automatically on dynamically loaded content

## Installation

1. Clone or download the repo:
   ```bash
   git clone https://github.com/yourusername/youtube-no-shorts.git
   ```
2. Open Firefox: `about:debugging#/runtime/this-firefox`
3. Click **Load Temporary Add-on** and select `manifest.json`
4. Enjoy a Shorts-free YouTube
